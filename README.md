# excel-data-analytics-dashboard-portfolio

This repository showcases a collection of interactive Excel dashboards developed to analyze sales performance, customer service operations, and business metrics using Microsoft Excel.

## Dashboards Included

### Coffee Shop Sales Dashboard
- Sales KPIs
- Product performance
- Store analysis
- Hourly sales trends

### Customer Support Dashboard
- Customer sentiment analysis
- SLA tracking
- Call center performance
- Business recommendations

### Vrinda Store Dashboard
- Orders vs Sales
- State-wise sales
- Gender analysis
- Channel performance

### FNP Sales Dashboard
- Revenue analysis
- Occasion-wise sales
- Customer spending
- Product category analysis

## Skills Demonstrated
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Interactive Dashboards
- KPI Reporting
- Business Analytics
